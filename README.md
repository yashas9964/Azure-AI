# Machine Learning Assignments

This repository contains implementation of basic Machine Learning algorithms as part of academic assignments.

## Assignment 1
- Linear Regression: Used to predict wine quality based on input features.
- Logistic Regression: Used to classify wine as good or bad.

## Assignment 2
- Decision Tree: Rule-based classification model.
- Random Forest: Ensemble of multiple decision trees.
- Support Vector Machine (SVM): Margin-based classifier.
- Principal Component Analysis (PCA): Dimensionality reduction technique.

## Dataset
- Wine Quality Dataset (WineQT.csv)
- Contains physicochemical properties like acidity, pH, alcohol, etc.

## Tools & Technologies
- Python
- Scikit-learn
- Google Colab

## Conclusion
These models demonstrate basic concepts of regression, classification, and dimensionality reduction in Machine Learning.
